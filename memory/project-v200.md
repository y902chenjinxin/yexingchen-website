- 前端架构: Vue3 + Composables（粒子系统/入场动画/降级分离）
- CSS变量: 14维度完整变量体系
- 性能降级: 三级降级（高/中/低）
- 音频驱动: Web Audio API AnalyserNode
- 后端变更: 无（localStorage存储）
