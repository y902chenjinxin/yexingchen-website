## 2. v1.2 仙侠风格升级

### 2.1 登录页重设计（仙侠风格）

#### 设计方向
- **Anchor**：Lo-Fi Luxury Hotel（仙侠版的典雅水墨风）
- **Palette**：墨白 + 青绿 + 淡金，不是紫蓝渐变
- **Background**：水墨山水层（CSS 多层渐变模拟云雾+远山）
- **门**：深木色 + 金线边框
- **光效**：青绿色灵气流动
- **卡片**：磨砂玻璃漂浮在云雾中，细金边框

#### 实现效果
- 深色背景 #0a0f14
- 水墨山水远山剪影
- 月光装饰
- 青绿色灵气光效（#7fffd4）
- 磨砂玻璃卡片 + 金边框
- 灵气流动动画

### 2.2 加载动画延长

#### 调整
- 原动画：200ms → 800ms → 1200ms → 2000ms
- 新动画：300ms → 900ms → 1500ms → 2500ms（延长1.5s）

### 2.3 岛屿重设计（仙侠浮空岛）

#### 设计方向
- 夜空背景 + 星辰 + 月光
- 岩石底座 + 植被 + 古风建筑
- 云雾缭绕在岛屿底部
- 立体感和悬浮感

#### 五岛屿主题色
- 音乐岛：古琴 + 枯木 + 青绿灵气（#7fffd4）
- 小说岛：书阁 + 竹简 + 金色灵气（#d29922）
- 视频岛：影视装置 + 胶片 + 紫色灵气（#a371f7）
- 日志岛：砚台 + 毛笔 + 银白灵气（#c9d1d9）
- 工具岛：齿轮 + 机关 + 橙色灵气（#f0883e）

## 3. 功能更新

### 3.1 备案信息展示

#### 实现
- 首页底部添加备案区
- 图标：public/filing/beian-icon.png
- 编号：
  - 皖公网安备34010402704746号
  - 皖ICP备2026006516号
- 链接：

  - 网络备案：https://beian.miit.gov.cn/#/Integrated/index

### 3.2 头像下拉菜单

#### 实现
- 右上角头像区域点击展开下拉菜单
- 菜单项：
  - 个人中心（开发中占位）
  - 修改密码（对话框）
  - 选择头像（随机2个图标：🌙 ☁️）
  - 岛屿导航（各岛跳转链接）
  - 管理后台（仅管理员）
  - 退出账号

#### 移除
- 底部固定"岛屿导航"按钮

### 3.3 背景音乐生效

#### 问题解决
- garden_music.wav 已上传到 /var/www/yexingchen/backend/uploads/bgm/
- 数据库 global_settings 已设置 bg_music key

### 3.4 RBAC 角色权限

#### 数据库变更
- users 表新增：
  - is_super_admin INTEGER DEFAULT 0（1=超级管理员）
  - is_test_user INTEGER DEFAULT 0（1=测试用户）

#### 权限校验
- require_super_admin 支持 is_super_admin=1 或 role='admin'
- admin@yexingchen.cn 已设为超级管理员

### 3.5 测试数据管理

#### 数据库变更
- 各内容表新增 is_test_data INTEGER DEFAULT 0
- 涉及表：music, novels, videos, tools

## 4. 技术架构

同 v1.1：
- **前端**：Vue 3 + Vite + Pinia + Vue Router + Element Plus
- **后端**：Python FastAPI + SQLAlchemy + JWT
- **数据库**：SQLite
- **文件存储**：腾讯 COS
- **邮件**：QQ邮箱 SMTP
- **部署**：Nginx + Uvicorn
