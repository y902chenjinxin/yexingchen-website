# PRD v2.11 - 水墨国风重构

## 1. 产品定位

**版本定位**：视觉重构 · 水墨国风
**版本目标**：将网站从"卡通游戏风"改为素雅禅意的传统水墨画意境

---

## 2. 功能详情

### 2.1 登录页重构

**背景**：当前登录页卡通门特效过于游戏化，缺乏国画韵味

**改动**：
- 去除冰晶琉璃门特效
- 改为淡墨山水长卷背景
- 登录卡片：宣纸质感 + 毛边效果
- 淡墨边框，藤黄聚焦效果

**验收标准**：
- [ ] 无卡通门特效
- [ ] 背景为水墨山水风格
- [ ] 卡片有宣纸质感

---

### 2.2 首页标题去除

**背景**：左上角"叶兴辰的个人网站"过于直白，破坏水墨意境

**改动**：
- 完全移除 `site-name` 元素

**验收标准**：
- [ ] 左上角无文字显示

---

### 2.3 首页玉简重构

**背景**：当前玉简金色描边过强，像游戏装备而非水墨卷轴

**改动**：
- 去除金色描边
- 改用淡墨线稿边缘
- 宣纸底色背景
- 聚焦态：淡墨晕染效果（非金色光晕）

**验收标准**：
- [ ] 玉简无金色边框
- [ ] 边缘为淡墨风格
- [ ] 聚焦态显示淡墨晕染

---

### 2.4 云雾层重构

**背景**：当前云雾透明度偏高，视觉太重

**改动**：
- MistLayer 透明度降至 0.4
- blur 强度加至 40px
- 整体更淡更透气，模仿留白

**验收标准**：
- [ ] 云雾几乎透明若隐若现
- [ ] 不遮挡玉简内容

---

### 2.5 整体配色重构

**背景**：当前大地色系过于"自然有机"，偏卡通

**改动**：
- 背景：深墨色 `#1a1a1a`
- 主色调：黛青 `#3d4f4f`
- 点缀：赭石 `#8b6b4a`、藤黄 `#c4a35a`
- 文字：淡墨 `#d8d4c8`
- 去除高饱和金色

**验收标准**：
- [ ] 背景为深墨色
- [ ] 主色调为低饱和黛青
- [ ] 金色使用极少

---

### 2.6 动画优化

**背景**：当前动画偏快，缺乏水墨的飘逸感

**改动**：
- 所有过渡动画：400ms → 800ms
- 云雾漂移周期：80s → 120s+
- 星空闪烁：18s → 24s
- 玉简悬浮动画：轻柔缓动

**验收标准**：
- [ ] 所有动画明显变慢
- [ ] 无快速弹跳效果

---

## 3. 技术方案

### 3.1 文件变更

| 文件 | 变更 |
|------|------|
| frontend/src/views/HomeView.vue | 移除 site-name 元素 |
| frontend/src/assets/styles/variables.css | 替换为水墨国风配色 |
| frontend/src/views/LoginView.vue | 水墨风格改造 |
| frontend/src/components/effects/MistLayer.vue | 透明度调低 |

### 3.2 配色变量替换

```css
/* 旧配色 */
--color-bg-dark: #E8DCC7;
--color-gold: #b8860b;

/* 新配色 */
--ink-black: #1a1a1a;
--dai-green: #3d4f4f;
--ochre: #8b6b4a;
--paper-white: #f5f2eb;
```

## 5. 踩坑预警

- 深色背景需确保文字对比度足够
- 黛青色饱和度极低，避免发灰发脏
- 动画降速后用户体验需确认无"卡顿"感
