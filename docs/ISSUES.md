### 技术方案摘要

| 模块 | 架构 | 关键决策 |
|------|------|----------|
| 前端架构 | Vue3 + Composables | 粒子系统/入场动画/降级分离 |
| CSS变量 | 14维度完整变量体系 | 全部可配置，禁止硬编码 |
| 性能降级 | 三级降级（高/中/低）| 自动检测 hardwareConcurrency |
| 音频驱动 | Web Audio API AnalyserNode | 低频→岛屿呼吸，高频→粒子密度 |
| 后端变更 | 无（localStorage存储）| 仅需扩展用户进度模型 |
