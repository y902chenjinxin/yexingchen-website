## 1. 需求概述

**背景**：用户反馈视觉效果不够高级、缺乏质感，需要对全站进行视觉升级。

**目标**：将网站从蓝灰调升级为"玄墨流金"设计系统，提升视觉质感和用户体验。

## 4. 技术方案

### 前端变更

**Design Token变更**：
- 新增 `--color-bg-dark: #0D0F14` 深墨背景
- 新增 `--color-gold` 古铜金色系
- 新增 `--color-qi-primary` 翡翠绿灵气色
- 强化多层阴影深度系统
- 添加 Grain噪点纹理

**组件接口**：
- LoginView.vue: TIMING常量调整
- HomeView.vue: CSS animation公转轨道
- ProfileView.vue: 样式迁移到CSS变量
- AdminView.vue: Element Plus深色覆盖层

**技术风险**：
- CSS变量覆盖层级需注意特异性
- 动画性能需验证50+岛屿元素

### 后端变更

- **无后端变更** — 纯前端视觉优化
