## 3. 功能详情

### 3.1 页面加载动画

**主题**：仙气灵蛇 + 星辰夜空

**设计**：
- 一条灵蛇从屏幕外蜿蜒游入，身体由半透明粒子光效构成
- 运动轨迹带有飘逸的粒子拖尾
- 背景：星辰夜空 + 云雾缭绕
- 整体：4K 电影质感，低饱和度青色

**动画流程**：
1. 灵蛇从左侧游入
2. 蜿蜒盘踞成圆形（进度环）
3. 进度环填充完毕，消散

**技术实现**：
- Canvas 粒子系统
- 贝塞尔曲线路径
- 60fps 动画

---

### 3.2 登录门动画

**主题**：冰晶琉璃门

**设计**：
- 两扇冰晶琉璃门，表面有缓慢流淌的蓝色光纹
- 门缝渗出寒气与星光
- 门以缓动曲线打开（先慢后快）
- 门边缘拖出粒子光尘
- 开门时有短暂阻力停顿（模拟封印挣脱）
- 全开后粒子风吹出 + 地面光环涟漪
- 整体画面轻微震动 + 空间扭曲波纹
- 然后登录弹窗缓缓出现

**动画时序**：
| 时间 | 效果 |
|------|------|
| 0s | 门静止，光纹流淌 |
| -0.2s | 门缝渗出寒气星光（相对于门开始打开前0.2s） |
| 0s | 光纹从外向内逐圈熄灭 |
| 0~1.5s | 门以缓动曲线打开（先慢后快，1.5s） |
| 1.5s | 阻力停顿 0.05s（模拟封印挣脱） |
| 1.55s | 瞬间加速到全开 |
| 1.6s+ | 粒子风吹出 + 光环涟漪 |
| 1.8s+ | 画面震动 + 扭曲波纹 |
| 2.0s | 登录弹窗出现 |

### 3.3 背景音乐

**问题**：现有音乐图标不够美观

**解决方案**：
- 新图标：音符图标或自定义图标
- 点击图标展开下拉列表
- 支持选择/切换背景音乐

**新增曲子**（调式相近，循环播放）：
1. 笛子独奏版（参考兰亭序调式）
2. 古琴弹奏版（参考青花瓷调式）

**技术实现**：

- 60秒单曲，循环播放
- 通过 `global_settings` 表管理音乐列表或前端固定配置

### 3.4 岛屿设计

**问题**：当前 SVG 扁平感强，不够立体

**视觉实现**：
- 仙侠风格 SVG 岛屿（5座：音乐、小说、视频、日志、工具）
- 透明背景，无边框
- 岩石底座 + 植被 + 古风建筑
- 云雾缭绕在岛屿底部

**动画效果（最终实现）**：
- 所有岛屿周转轨道运动（不是简单的上下浮动）
- 每座岛屿独立的椭圆轨迹，周期 12-18s
- 悬停时岛屿上浮 + 放大 + 光晕增强

| 岛屿 | 轨迹周期 | 幅度 |
|------|---------|------|
| 音乐岛 | 12s | ±8px, ±12px, ±2° |
| 小说岛 | 15s | ±10px, ±15px, ±2.5° |
| 视频岛 | 18s | ±6px, ±10px, ±1.5° |
| 日志岛 | 14s | ±12px, ±14px, ±3° |
| 工具岛 | 16s | ±9px, ±13px, ±2° |

**动画 CSS**：
```css
.music-island { animation: orbit-music 12s linear infinite; }
@keyframes orbit-music {
  0% { transform: translate(0, 0) rotate(0); }
  25% { transform: translate(8px, -12px) rotate(2deg); }
  50% { transform: translate(0, -20px) rotate(0); }
  75% { transform: translate(-8px, -12px) rotate(-2deg); }
  100% { transform: translate(0, 0) rotate(0); }
}
```

### 3.5 个人中心

#### 3.5.1 信息管理

**功能**：
- 查看自己的基础信息（昵称、邮箱、头像、注册时间）
- 修改昵称
- 选择头像

**API**：
- `GET /api/users/me` - 获取当前用户信息
- `PUT /api/users/me` - 更新昵称、头像

#### 3.5.2 角色权限管理

**功能**：
- 管理员可查看所有用户
- 管理员可增删改用户角色
- 角色：super_admin, admin, user, test_user

**页面**：
- 用户列表（分页）
- 用户详情弹窗（查看/编辑）
- 角色分配下拉

**API**：
- `GET /api/admin/users` - 获取用户列表（需 admin 权限）
- `PUT /api/admin/users/{id}/role` - 修改用户角色
