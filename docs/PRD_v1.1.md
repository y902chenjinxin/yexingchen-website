## 2. v1.1 视觉与体验升级

### 2.1 登录页重设计

#### 问题（v1.0）
- 背景为深青墨色，色调沉闷，不够年轻化
- 光效为金色流光，视觉冲击力不足，缺乏惊艳感
- 开门效果平淡，缺少"洞天初开"的仪式感

#### 解决方案（v1.1）

**背景设计**
- 采用年轻化渐变配色：紫色(#667eea) → 粉紫(#764ba2) → 深蓝(#1a1a2e)
- 添加光斑装饰：右上角和左下角添加柔和的白色径向渐变光斑
- 整体氛围：年轻、活力、神秘感

**光效优化**
- 纯净白光替代金色光效
- 光柱角度调整为-15度
- 添加模糊效果(blur: 2px)使光线更柔和自然
- 光柱宽度收窄至25%，高度延伸至300%
- 动画时长延长至1000ms，曲线优化为cubic-bezier(0.25, 0.46, 0.45, 0.94)

**开门效果增强**
- 木门颜色从深棕(#3D2517)改为深蓝黑(#1a1a2e)，边框改为白色半透明
- 门边框：4px solid rgba(255,255,255,0.4)
- 开门瞬间增加「光芒绽放」效果：白色径向渐变从中心向外扩散
- 光球动画：800ms ease-out，从scale(0.8)到scale(1.5)，透明度从1到0

**配色调整**
- 登录卡片背景：rgba(30, 30, 60, 0.85)
- 卡片边框：1px solid rgba(255,255,255,0.2)
- 卡片阴影：增加紫色光晕 0 0 80px rgba(102, 126, 234, 0.15)
- 标题文字：白色 + 紫色文字阴影
- 强调色：#a78bfa（淡紫色）

### 2.2 主页面背景重设计

#### 问题（v1.0）
- 背景为深青渐变，缺乏天空感和年轻感
- 云海效果较弱，装饰性不足

#### 解决方案（v1.1）

**渐变蓝天背景**
```
从浅蓝 #87CEEB → 天蓝 #B0E0E6 → 淡蓝 #E0F0FF → 极浅蓝 #F5F9FF
```

**云朵装饰**
- 在底部（85%-95%高度）添加3个大面积白色椭圆云
- 透明度：60%-80%
- 尺寸：250px-400px宽度，100px-150px高度
- 无动画，保持静态装饰感

**岛屿卡片配色**
- 卡片背景：白色半透明渐变 rgba(255,255,255,0.95)
- 文字颜色：深色 #1a1a2e（替代原来的浅色文字）
- 岛屿名称字号：20px
- 副标题：#4a5568（深灰）
- 强调色：#667eea（紫色）

**顶栏配色**
- 网站名称：深色 + 白色文字阴影
- 音乐控制按钮：紫色边框和背景

**底部切换栏**
- 背景：白色半透明 rgba(255,255,255,0.95)
- 边框：2px solid rgba(102,126,234,0.3)
- 文字：紫色 #667eea
- 悬停：边框变为纯紫色，增加光晕

**岛屿弹层**
- 背景：白色 rgba(255,255,255,0.98)
- 边框：2px solid rgba(102,126,234,0.3)
- 项目悬停背景：rgba(102,126,234,0.1)
- 分隔线：rgba(102,126,234,0.2)

### 2.3 岛屿视觉升级

#### 问题（v1.0）
- 岛屿仅使用emoji图标（🎵📖🎬📝⚙️），视觉层次低
- 缺乏「云上浮岛」的设计感和沉浸感

#### 解决方案（v1.1）

为每个岛屿设计独立的SVG插图作为视觉主体：

**音乐岛** (music-island.svg)
- 主题色：天蓝色清新天空
- 元素：古琴(guqin)、竹林、白云飘动
- 细节：古琴琴弦、古琴木纹色调、竹叶点缀
- 浮动音符装饰 ♪♫

**小说岛** (novel-island.svg)
- 主题色：温暖的米白色/羊皮纸色
- 元素：展开的书卷/竹简、翠绿竹林
- 细节：书页上的文字线、古典印章
- 飘浮汉字：書卷典文

**视频岛** (video-island.svg)
- 主题色：深蓝紫色电影感夜空
- 元素：胶片卷、电影放映机、放映之光
- 细节：胶片孔洞、星星点缀、播放按钮金光
- 顶部装饰：胶片条

**日志岛** (log-island.svg)
- 主题色：墨绿色静谧氛围
- 元素：竹签筒、木牌记录、墨笔
- 细节：木牌上的墨线、红绳悬挂
- 装饰：毛笔、红色印章

**工具岛** (tool-island.svg)
- 主题色：深灰工业金属质感
- 元素：齿轮组、黄铜齿轮、银色零件
- 细节：齿轮咬合、工具投影（小扳手、螺丝刀）
- 背景：微弱电路纹理

**岛屿卡片尺寸调整**
- 宽度：200px → 220px
- 高度：220px → 260px
- 图片尺寸：180x180px（contain模式）
