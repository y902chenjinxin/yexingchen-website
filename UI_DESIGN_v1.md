# 个人网站 UI 设计初稿 v1.0

> 基于 PRD v1.0，本文档为视觉设计初稿，待评审确认后进入开发。

---

## 1. 设计规范

### 1.1 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #1A3A4A | 深青墨，主背景 |
| 次色 | #2D5A6B | 青蓝，卡片/岛屿底色 |
| 强调色 | #4ECDC4 | 碧青，高亮/按钮 |
| 金色点缀 | #C9A962 | 仙侠元素点缀（云边缘光效） |
| 文字主色 | #E8F4F0 | 浅青白，正文 |
| 文字次色 | #8BA5B5 | 灰青，辅助文字 |
| 背景渐变 | #0D1F27 → #1A3A4A | 顶部深到底部渐变 |
| 云色 | rgba(255,255,255,0.08) | 半透明白，浮云元素 |
| 危险色 | #E85A5A | 删除/错误 |
| 成功色 | #4ECDC4 | 成功提示 |

### 1.2 字体

| 用途 | 字体 | 说明 |
|------|------|------|
| 标题 | "Noto Serif SC", "Source Han Serif SC", serif | 仙侠古风 |
| 正文 | "Noto Sans SC", "Source Han Sans SC", sans-serif | 清晰易读 |
| 英文/数字 | "Inter", sans-serif | 现代感 |

### 1.3 动效规范

| 动效 | 时长 | 曲线 | 说明 |
|------|------|------|------|
| 页面渐入 | 600ms | ease-out | 元素淡入 |
| 岛屿浮动 | 6s | ease-in-out infinite | 上下浮动 8px |
| 云飘动 | 20s | linear infinite | 背景云层平移 |
| 门户开 | 1200ms | cubic-bezier(0.65, 0, 0.35, 1) | 双门对称展开 |
| 光效闪过 | 800ms | ease-in-out | 横向流光 |

### 1.4 响应式断点

| 设备 | 宽度 |
|------|------|
| 桌面 | ≥1024px |
| 平板 | 768px ~ 1023px |
| 手机 | < 768px |

## 2. 登录页设计

### 2.1 动画时序

```
0ms      → 页面黑屏/深青背景，云雾缭绕
0-800ms  → 金色流光从左向右划过（光宽约占屏幕30%）
800-2000ms → 云雾向两侧散开
1200-2400ms → 古风木门双开（门缝从0到全开）
2000-2600ms → 登录卡片淡入+上浮（translateY: 20px→0）
```

### 2.2 登录卡片布局

```
┌────────────────────────────────┐
│                                │
│         【云雾/木门背景】          │
│                                │
│    ┌──────────────────────┐    │
│    │      叶兴辰的个人网站     │    │ ← 标题，Noto Serif SC
│    │        subtitle       │    │ ← 副标题：神农遗风，云上洞天
│    │                      │    │
│    │  邮箱输入框  [获取验证码] │    │
│    │  验证码输入框            │    │
│    │  密码输入框             │    │
│    │                      │    │
│    │  [登 录]  [注册申请]   │    │
│    │                      │    │
│    │    忘记密码？          │    │
│    └──────────────────────┘    │
│                                │
└────────────────────────────────┘
```

- 卡片：`background: rgba(26, 58, 74, 0.85); backdrop-filter: blur(12px);`
- 边框：`1px solid rgba(78, 205, 196, 0.3);`
- 圆角：12px
- 阴影：`0 8px 32px rgba(0,0,0,0.4);`

### 2.3 门的设计

- 样式：双扇古风木门，每扇门居中，各占50%宽度
- 门板颜色：深木色 #3D2517，纹路用 CSS 渐变模拟
- 门框：金色边框 #C9A962，4px
- 门上可以有简化云纹/山水浮雕（CSS 或 SVG）

### 2.4 光效设计

- 一道弧形光带，从左下斜向右上方
- 渐变：`rgba(201, 169, 98, 0.8)` → `rgba(255, 255, 255, 0.4)` → 透明
- 宽度：约占视口 30%
- 拖尾效果：模糊边缘

## 3. 主页面设计

### 3.1 整体布局

```
┌─────────────────────────────────────────────────┐
│  [顶栏] Logo / 用户名 / 背景音乐开关 / 设置      │
├─────────────────────────────────────────────────┤
│                                                 │
│           ☁   ☁                                │
│      ☁        ☁   ☁                           │
│                                                 │
│   【音乐岛】    【小说岛】                       │
│      🎵          📖                            │
│              ☁                                 │
│                    【视频岛】                   │
│                      🎬                        │
│        ☁                          ☁            │
│                    【日志岛】                  │
│                      📝                        │
│   ☁                                      ☁    │
│                      【工具岛】                 │
│                        ⚙                      │
│                                                 │
├─────────────────────────────────────────────────┤
│  [底栏]  ↑ 返回顶部  |  当前岛屿名              │
└─────────────────────────────────────────────────┘
```

### 3.2 岛屿浮动效果

每个岛屿有轻微的上下浮动动画，错开相位避免视觉疲劳：

```css
/* 音乐岛 */
animation: float-island 6s ease-in-out infinite 0s;

/* 小说岛 */
animation: float-island 6s ease-in-out infinite 1.2s;

/* 视频岛 */
animation: float-island 6s ease-in-out infinite 2.4s;

/* 日志岛 */
animation: float-island 6s ease-in-out infinite 3.6s;

/* 工具岛 */
animation: float-island 6s ease-in-out infinite 4.8s;

@keyframes float-island {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-8px); }
}
```

### 3.3 岛屿视觉样式

每个岛屿是一个圆角矩形卡片，悬于云海之上：

```
┌──────────────────┐
│   ☁  云雾环绕    │  ← 半透明白色云雾环绕岛屿边缘
│                  │
│      🎵          │  ← 大图标（SVG图标）
│                  │
│   【音乐岛】      │  ← 岛屿名称
│    音律飘渺        │  ← 副标题
│                  │
│    点击进入 >>    │  ← 提示文字
└──────────────────┘
```

- 岛屿尺寸：桌面 200x220px，平板 160x180px，手机 140x160px
- 背景：渐变青 + 各岛屿专属图标色点缀
- 岛屿底部有倒影：`filter: blur(4px) opacity(0.3) scaleY(-0.5);`
- 悬停效果：放大 1.05，云雾加浓，微微发光

### 3.4 岛屿专属配色

| 岛屿 | 主色 | 强调色 | 图标 |

## 4. 各岛屿空间设计

### 4.1 通用布局

```
┌─────────────────────────────────────────────────┐
│  [顶栏]  ← 返回主岛  |  【岛屿名】  |  搜索框     │
├─────────────────────────────────────────────────┤
│                                                 │
│   【工具栏：上传/筛选/排序】                      │
│                                                 │
│   ┌────────┐  ┌────────┐  ┌────────┐           │
│   │ 项目1  │  │ 项目2  │  │ 项目3  │           │
│   │  封面  │  │  封面  │  │  封面  │           │
│   │  标题  │  │  标题  │  │  标题  │           │
│   │标签/分类│  │标签/分类│  │标签/分类│           │
│   │ [下载] │  │ [下载] │  │ [下载] │           │
│   └────────┘  └────────┘  └────────┘           │
│                                                 │
│   ┌────────┐  ┌────────┐  ┌────────┐           │
│   │ 项目4  │  │ 项目5  │  │ 项目6  │           │
│   │ ...    │  │ ...    │  │ ...    │           │
│   └────────┘  └────────┘  └────────┘           │
│                                                 │
└─────────────────────────────────────────────────┘
```

- 网格布局：桌面 4列，平板 3列，手机 2列
- 内容卡片圆角 12px，悬停上浮+阴影加深

### 4.2 各岛屿特殊设计

**音乐岛卡片**：
- 封面：专辑封面图（默认灰色音律图标占位）
- 显示：标题、时长、格式、分类标签
- 操作：播放（播放按钮弹出播放器）、下载

**小说岛卡片**：
- 封面：小说封面图（默认竹简图标占位）
- 显示：书名、作者、分类标签
- 操作：下载（epub直接下载）

**视频岛卡片**：
- 封面：视频封面图（默认胶片图标占位）
- 显示：标题、腾讯COS链接（可点击复制）、分类标签
- 操作：复制链接、下载封面

**日志岛**：
- 无卡片列表，纯时间线布局
- 每条日志：`[时间] [操作者] [操作类型] [详情]`
- 管理员可看到所有用户日志，普通用户只能看自己

**工具岛卡片**：
- 显示：标题、链接（图标+域名）、添加时间
- 操作：点击新窗口打开、编辑、删除

## 7. 关键动效示例代码（CSS）

### 7.1 登录页光效闪过
```css
.light-effect {
  position: absolute;
  width: 30%;
  height: 200%;
  top: -50%;
  left: -30%;
  background: linear-gradient(90deg, transparent, rgba(201,169,98,0.8), rgba(255,255,255,0.4), transparent);
  transform: rotate(-15deg);
  animation: light-sweep 800ms ease-in-out forwards;
}

@keyframes light-sweep {
  0% { left: -30%; opacity: 0; }
  50% { opacity: 1; }
  100% { left: 130%; opacity: 0; }
}
```

### 7.2 门户双开
```css
.door {
  position: absolute;
  width: 50%;
  height: 100%;
  background: linear-gradient(180deg, #3D2517 0%, #2A1A0F 100%);
  border: 4px solid #C9A962;
  transition: transform 1200ms cubic-bezier(0.65, 0, 0.35, 1);
}

.door-left {
  left: 0;
  border-right: 2px solid #C9A962;
  transform-origin: left center;
}

.door-right {
  right: 0;
  border-left: 2px solid #C9A962;
  transform-origin: right center;
}

.doors-open .door-left { transform: scaleX(0); }
.doors-open .door-right { transform: scaleX(0); }
```

### 7.3 云雾散开
```css
.cloud {
  position: absolute;
  width: 100%;
  height: 100%;
  background: radial-gradient(ellipse at center, rgba(255,255,255,0.15) 0%, transparent 70%);
  transition: opacity 1000ms ease-out, transform 1000ms ease-out;
}

.cloud-left {
  transform-origin: left center;
}

.cloud-right {
  transform-origin: right center;
}

.clouds-open .cloud { opacity: 0; }
.clouds-open .cloud-left { transform: translateX(-100%); }
.clouds-open .cloud-right { transform: translateX(100%); }
```
